ZoneGuard EV - Professional Dashboard Cleanup

Changes in this package:
1. Replaced demo/game-style quick action buttons with a clean academic workflow:
   - Simulation Control: Connect + Spawn, Start Normal Drive, Reset Scenario
   - Driver Control Mode: Autonomous Driving, Human Keyboard Control
   - Attack Execution: Apply Selected Attack
   - ZoneGuard Defense: Train Baseline, Run Adaptive Recovery, Emergency Stop
2. Removed the visible top quick buttons for: Run Full Demo, Urban Attack, Highway Attack, Sensor Failure, Mixed Emergency, Random Attack, Start Live Loop, Attack Vehicle Now, Reset Demo.
3. Kept the underlying advanced endpoints in code for compatibility, but they are no longer shown in the main top control panel.
4. Added top-level bindings for the new workflow buttons.
5. Updated cache-busting version to professional-clean-1.

After extracting this ZIP:
- Run from the new extracted folder, not an older copy.
- Open http://127.0.0.1:5000
- Press Ctrl+F5 or use Incognito to avoid old browser cache.
