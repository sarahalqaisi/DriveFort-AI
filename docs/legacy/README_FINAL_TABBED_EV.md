# ZoneGuard EV Final Tabbed Platform

This final build removes EV branding from the UI and uses a neutral Electric Vehicle (EV) identity.

## UI Tabs
- Driver / Owner View
- Attacker Console
- ZoneGuard Defense
- Evidence & Replay

## Run
```bat
run_carla_windows.bat
```

Recommended demo flow:
1. Connect + Spawn
2. Start Normal Drive
3. Open Attacker Console tab and apply an attack
4. Open ZoneGuard Defense tab and activate protection/recovery
5. Open Evidence & Replay tab and generate/report evidence
