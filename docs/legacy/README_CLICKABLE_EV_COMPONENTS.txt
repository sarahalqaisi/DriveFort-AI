ZoneGuard EV - Clickable EV Components Update

This version adds an interactive Component Detail Panel to the dashboard.

How to use:
1. Open the dashboard.
2. Go to the EV Internal Systems / Smooth Vehicle Tracking panel.
3. Click any component label such as Motor, Brakes, Steering, Battery, Inverter, Sensors, GPS, HMI, or ECU/CAN.
4. The Component Detail Panel will show:
   - subsystem structure
   - function
   - related attacks
   - cyber impact
   - ZoneGuard defense action
   - live component status: Normal / Warning / Critical / Recovered

During attacks, affected components are highlighted and the detail panel reflects the current risk state.
